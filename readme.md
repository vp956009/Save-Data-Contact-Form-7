== Description ==

**Contact Form 7 Save Data** allow to save in db with particular form where you need to save any record of database. Contact form 7 not provide to store data in database so we make **contact form 7 save to database plugin** in to in your [WordPress](https://en.wikipedia.org/wiki/WordPress) site.

[Demo](http://oceanwebguru.com/oceanwebdemo/contact-form-7-save-data-demo/)

> **Plugin Features**
>
> * Save Data to database with contact form 7
> * Make admin side page show contact form 7 to enable option to save
> * Export Record With CSV
> * Can be view individual record
> * Easy to use
